Bright Coffee Shop Sales Analysis – 2026 Case Study Project Overview

This project was completed as part of the BrightLearn Data Analytics Case Study (2026). The objective of the project was to analyse historical sales data from a coffee shop and extract meaningful insights that can help a new CEO improve revenue and product performance.

The analysis focuses on identifying top-performing products, understanding sales patterns across different times of the day, and providing data-driven recommendations for business growth.

Business Objective

The main goal of this project is to use data analytics and SQL to help the business understand:

Which products generate the most revenue

What time of day the store performs best

Sales trends across different product categories

Opportunities to improve sales performance

Tools Used

The following tools and platforms were used during the project:

Miro (Project planning and data flow diagram)

Microsoft SQL Server / Databricks (Data processing and SQL analysis)

Microsoft Excel (Data analysis, pivot tables, and charts)

PowerPoint (Final presentation)

GitHub (Project submission)

Project Workflow

Planning & Architecture
A project plan was created in Miro to show:

Where the data comes from

How the data is processed (ETL process)

Where the data is stored

How the data is analysed and presented

Data Processing
The dataset was cleaned and prepared before analysis:

Excel file converted to CSV

Data uploaded to Databricks

Data cleaned and formatted

Created new calculated column:

total_amount = unit_price × transaction_qty

Created time buckets (30-minute intervals)

Extracted day, month, and hour from the transaction time

SQL Analysis
SQL queries were written to analyse the dataset and extract insights such as:

Total revenue by product type

Total revenue by product category

Revenue by hour of the day

Revenue by 30-minute intervals

Best-selling products

Worst-performing products

Total quantity sold per product

Data Visualisation
The cleaned dataset was exported to Excel and used to create:

Revenue by product category charts

Sales by time of day charts

Monthly sales trend charts

Best-selling product visualisations

Pivot tables for deeper analysis

Key Insights
The analysis helped identify:

The products that generate the highest revenue

The busiest sales hours during the day

Underperforming products

Sales patterns and trends

Opportunities to improve sales performance

Recommendations
Based on the analysis, the following recommendations were provided:

Promote high-performing products

Improve or replace low-performing products

Run promotions during slow sales hours

Increase stock for best-selling items

Use sales trends to support future business decisions
